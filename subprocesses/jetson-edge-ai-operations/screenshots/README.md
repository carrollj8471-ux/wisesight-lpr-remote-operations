# Screenshots

Screenshots are optional and must be redacted before committing.

Good examples:

- `uname -r` showing a Jetson kernel such as `5.10.192-tegra`
- Clean `dpkg --audit` output
- Held `nvidia-l4t-*` packages
- `teamviewerd` running with TeamViewer ID hidden
- Jetson UEFI boot screen with serials hidden
